Test Case 3: Edge Positioning

Scroll down and right-click near the bottom or right edge of the screen to test menu positioning logic.

Hold Shift and right-click here to see the native browser context menu instead.

Test Case 5: Close Menu

Right-click to open menu, then:
  • Press ESC key to close
  • Click elsewhere to close
  • Click a menu item to close and trigger action

Test Case 6: Performance

Rapidly right-click multiple times in different locations. Check browser console for performance metrics (should be <10ms).
